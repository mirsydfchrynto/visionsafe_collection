LAPORAN PENYELESAIAN TUGAS BIG DATA - VISIONSAFE

Berikut adalah penjelasan teknis untuk setiap tahapan pengerjaan sesuai dengan poin penilaian:

1. Data Collection (GitHub Actions & Google Colaboratory)
   - Metode: Crawling data berita kesehatan mata secara otomatis dari berbagai sumber otoritas global dan lokal (WHO, NIH, Kemenkes, Science Daily, dll).
   - Platform: Menggunakan GitHub Actions sebagai sistem "Autonomous Worker" yang menjalankan script Python (data_ingestion.py) secara terjadwal setiap hari.
   - Keunggulan: Sistem menggunakan "Surgical Extraction" (BeautifulSoup4 & lxml) untuk mengambil konten berita secara utuh, bukan hanya cuplikan RSS.

2. Data Storage (NoSQL - Firebase Firestore)
   - Database: Menggunakan Firebase Firestore sebagai penyimpanan NoSQL.
   - Keamanan: Data disimpan secara aman dengan skema koleksi 'visionsafe_knowledge'.
   - Deduplikasi: Implementasi hashing SHA-256 pada URL artikel sebagai Document ID unik untuk memastikan tidak ada redundansi data (Global Deduplication).

3. Data Preparation (Google Colaboratory)
   - Proses: Data ditarik dari Firestore ke Google Colab dalam format DataFrame (Pandas).
   - Preprocessing NLP:
     - Case Folding: Menyeragamkan teks menjadi huruf kecil.
     - Cleaning: Menghapus simbol, angka, dan karakter non-ASCII (Ghost characters).
     - Bilingual Stopword Removal: Menghapus kata sambung bahasa Indonesia (Sastrawi) dan bahasa Inggris (NLTK) untuk mendapatkan kata kunci yang berkualitas.
     - Filtering: Menggunakan "Smart Keyword Filter" untuk menjamin relevansi data terhadap topik kesehatan mata.

4. Analysis Data & Visualization (Google Colaboratory)
   - Visualisasi 1 (Big Data Variety): Menampilkan sebaran sumber data menggunakan bar chart horizontal yang rapi untuk menunjukkan keberagaman sumber berita (Multi-Source Intelligence).
   - Visualisasi 2 (WordCloud): Menampilkan tren riset kesehatan mata global (Global Trends) melalui visualisasi kata kunci yang paling sering muncul (seperti Miopia, Cataract, Screen Time).
   - Insight: Memberikan gambaran bagi pengguna aplikasi mengenai isu kesehatan mata apa yang sedang tren dan sumber mana yang paling aktif memberikan edukasi.
